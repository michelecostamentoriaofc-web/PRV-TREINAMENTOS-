# PRV Training Platform

Global training platform for live classes, on-demand courses, certificates, payments, companies, instructors, students and owner administration.

## Main Modules

- Public Website
- Student Portal
- Company Portal
- Instructor Portal
- Owner Dashboard
- Live Classes
- On-Demand Courses
- Certificates
- Payments
- In-Person Training
- Marketplace
- Global Catalog
- Compliance
- Reports

## Owner Access

The Owner Dashboard is exclusive to Michele Costa.

## Technology

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- LiveKit
- Stripe / Mercado Pago
